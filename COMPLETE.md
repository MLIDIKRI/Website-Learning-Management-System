## 🗄️ DATABASE STRUCTURE

```
14 Tables:
├── users (Admin, Teacher, Student)
├── classes (Course sections)
├── class_members (Student enrollment)
├── materials (Course materials)
├── assignments (Tasks/assignments)
├── assignment_submissions (Student work)
├── announcements (Class announcements)
├── discussions (Discussion topics)
├── discussion_comments (Replies)
├── attendance (Attendance records)
├── grades (Student grades)
└── admin_logs (Activity logging)

+ 3 additional system tables
```
